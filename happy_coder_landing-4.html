Frequently asked questions

How secure is the encryption?

End-to-end encryption with keys generated on your device. We literally cannot decrypt your conversations.

Does this work with my existing setup?

Yes. Run "happy" instead of "claude" and everything else stays the same. Your workflow, API keys, projects.

What about enterprise compliance?

Client-side architecture means your code never touches our servers. Ideal for strict security requirements.

Is it really free?

Core app is completely free. Open source means you can verify. You only pay for Claude API usage.

Which devices are supported?

iOS, Android, and web. Works on phones, tablets, foldables, whatever you have.

How does billing work?

Happy Coder uses Claude Code however you're logged in. Your API keys, your billing, your control.
